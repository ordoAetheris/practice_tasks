# Подсказки для тестирования: CouponService

## Обязательно протестировать
- Создание купона и проверка валидности
- Применение — цена уменьшается на процент
- Просроченный купон — исключение
- Лимит исчерпан — исключение
- Неподходящая категория — исключение
- Деактивация

## Продвинутые кейсы
- Пустой applicableCategories = все категории
- Скидка 100%
- Некорректный процент (0, 101, -1)
- Граничные даты
