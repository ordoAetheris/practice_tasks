# Подсказки для тестирования: EmployeeDirectoryService

## Обязательно протестировать
- Приём сотрудника — корректные поля
- Приём с отрицательной зарплатой — ошибка
- Увольнение — fired = true
- Увольнение уже уволенного — ошибка
- Перевод в другой отдел — department обновлён
- Перевод уволенного — ошибка
- Изменение зарплаты
- Поиск по отделу — возвращает только активных
- Средняя зарплата по отделу
- Сотрудники со стажем > N лет

## Продвинутые кейсы
- Изменение зарплаты уволенного — ошибка
- Пустой отдел — средняя зарплата 0
- Стаж ровно N лет — включается или нет (граничный случай)
- Несколько сотрудников в разных отделах — поиск возвращает только нужный отдел
- Уволенные не входят в расчёт средней зарплаты
