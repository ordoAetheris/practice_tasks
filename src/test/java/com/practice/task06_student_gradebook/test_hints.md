# Подсказки для тестирования: GradebookService

## Обязательно протестировать
- Добавление студента и предмета
- Выставление оценки — успешный сценарий
- Оценка вне диапазона (0, 6, -1) — ошибка
- Оценка несуществующему студенту или по несуществующему предмету — ошибка
- Средний балл студента с несколькими оценками
- Средний балл по предмету
- Топ-N студентов — корректная сортировка
- Получение оценок студента по предмету — отсортированы по дате

## Продвинутые кейсы
- Студент без оценок — средний балл 0.0
- Топ-N где N больше количества студентов
- Несколько оценок по одному предмету у одного студента
- Одинаковый средний балл у нескольких студентов в топе
- Добавление дублирующего студента/предмета — ошибка
