# Подсказки для тестирования: OrderManagementService

## Обязательно протестировать
- Полный жизненный цикл: CREATED -> PAID -> SHIPPED -> DELIVERED
- Нельзя пропустить статус (CREATED -> SHIPPED — исключение)
- Отмена из CREATED, PAID, SHIPPED — успешно
- Отмена доставленного — исключение
- Подсчёт суммы заказа (quantity * price по всем items)
- Поиск по клиенту и по статусу

## Продвинутые кейсы
- Повторная оплата уже оплаченного
- Заказ с пустым списком товаров
- Несколько заказов одного клиента в разных статусах
- Отмена уже отменённого заказа
