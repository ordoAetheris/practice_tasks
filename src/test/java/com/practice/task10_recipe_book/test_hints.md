# Подсказки для тестирования: RecipeBookService

## Обязательно протестировать
- Добавление и удаление рецепта
- Поиск по названию (подстрока, регистронезависимый)
- Поиск по ингредиенту
- Фильтр по сложности (EASY/MEDIUM/HARD)
- Фильтр по времени приготовления (<= заданного)
- Пустой результат при отсутствии совпадений

## Продвинутые кейсы
- Рецепты с пересекающимся набором ингредиентов
- Поиск рецептов, для которых есть все ингредиенты из заданного списка
- Несколько рецептов с одинаковым названием
- Граничные значения времени приготовления (0, MAX)
