# Подсказки для тестирования: InMemoryFileSystem

## Обязательно протестировать
- mkdir + ls: созданная директория видна в списке
- mkdirp создаёт промежуточные директории
- mkdir без родительской директории бросает исключение
- createFile + readFile: содержимое корректно
- writeFile перезаписывает содержимое
- delete удаляет файлы и директории рекурсивно
- find с шаблоном * находит файлы

## Продвинутые кейсы
- find с шаблоном ? совпадает ровно с одним символом
- mv перемещает файл, старый путь недоступен
- cp копирует директорию рекурсивно (глубокая копия)
- du суммирует размеры всех файлов рекурсивно
- createFile в несуществующую директорию бросает исключение
- readFile на директории бросает исключение
- Попытка удалить корневую директорию / — обработка ошибки
- Пути с // или trailing slash нормализуются
- ls на пустой директории возвращает пустой список
