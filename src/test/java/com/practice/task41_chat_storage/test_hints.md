# Подсказки для тестирования: ChatStorageService

## Обязательно протестировать
- Создание чата — проверить что возвращается объект с корректным списком участников
- Отправка сообщения — проверить статус SENT и корректные поля
- Отправка не-участником чата — ожидать IllegalArgumentException
- Редактирование сообщения автором — проверить новый текст и флаг edited=true
- Редактирование чужого сообщения — ожидать исключение
- Удаление сообщения — проверить что удалённое не возвращается в getMessages
- Пагинация getMessages — корректные limit и offset
- Поиск searchMessages — проверить поиск подстроки без учёта регистра
- markAsRead — проверить что getUnreadCount возвращает 0 после вызова
- getUnreadCount — корректный подсчёт для разных пользователей

## Продвинутые кейсы
- Создание чата с менее чем 2 участниками — ожидать ошибку
- Удаление уже удалённого сообщения
- Поиск по пустому запросу
- Пагинация с offset больше количества сообщений — пустой список
- Несколько чатов одновременно — изоляция данных
- Редактирование удалённого сообщения — ожидать ошибку
- Подсчёт непрочитанных учитывает только сообщения других участников
