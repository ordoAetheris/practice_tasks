# Подсказки для тестирования: PriceCalculatorService

## Обязательно протестировать
- PERCENTAGE скидка — 10% от 100 = 90
- FIXED скидка — скидка 15 от 100 = 85
- BUY_N_GET_M — купи 2, получи 1 бесплатно (3 товара по цене 2)
- Нестекаемые скидки — берётся лучшая, а не обе
- Стекаемые скидки — применяются последовательно
- findBestDiscount — возвращает скидку с максимальной выгодой
- calculateTotal — корректный расчёт налога
- Цена не ниже 0 — скидка больше цены => цена = 0

## Продвинутые кейсы
- minQuantity не достигнут — скидка не применяется
- Скидка для конкретного товара не применяется к другому
- BUY_N_GET_M с количеством меньше N — скидка не применяется
- BUY_N_GET_M с количеством = 2*N+M — два раза применяется
- Комбинация stackable и non-stackable скидок
- Пустой список товаров — итого = 0
- Пустой список скидок — полная цена
- BigDecimal точность — проверить округление (2 знака после запятой)
