# Подсказки для тестирования: TimeSeriesDB

## Обязательно протестировать
- write + query: записанные точки возвращаются в правильном порядке
- query по диапазону: точки вне диапазона не возвращаются
- queryWithTags фильтрует корректно (все указанные теги должны совпасть)
- aggregate AVG, SUM, MIN, MAX, COUNT — проверить каждую функцию
- downsample корректно разбивает на интервалы и агрегирует
- getMetrics возвращает все уникальные метрики

## Продвинутые кейсы
- Пустой диапазон (нет данных) — aggregate и downsample обрабатывают корректно
- Множество метрик не влияют друг на друга
- Даунсэмплинг: интервал больше всего диапазона — одна точка
- Даунсэмплинг: интервал меньше разницы между точками — пустые интервалы пропускаются
- Большое количество точек — производительность range-запроса
- Точки с одинаковым timestamp обрабатываются корректно
- queryWithTags с пустым tags — возвращает все точки в диапазоне
