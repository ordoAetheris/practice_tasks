# Подсказки для тестирования: LibraryService

## Обязательно протестировать
- Добавление книги и поиск по автору/жанру/названию
- Выдача книги — успешный сценарий
- Выдача книги — нет свободных экземпляров (все на руках)
- Возврат книги — успешный сценарий
- Возврат книги — повторный возврат (уже возвращена)
- Получение доступных книг (с учётом выданных и возвращённых)
- Получение просроченных выдач
- Поиск по названию — подстрока без учёта регистра
- Попытка выдать книгу несуществующему пользователю

## Продвинутые кейсы
- Несколько экземпляров одной книги: выдать totalCopies экземпляров, следующая выдача — ошибка
- Возврат одного экземпляра — книга снова доступна
- Просрочка: dueDate = сегодня — ещё не просрочена, dueDate = вчера — просрочена
- Выдача одной книги нескольким пользователям (пока есть экземпляры)
- Пустая библиотека — все методы возвращают пустые списки
