# Подсказки для тестирования: TaskExecutorService

## Обязательно протестировать
- Submit Runnable, задача выполняется
- Submit Callable, Future возвращает результат
- Несколько задач выполняются параллельно (на разных потоках)
- shutdown() ждёт завершения задач в очереди
- shutdownNow() прерывает текущие задачи
- getActiveCount и getQueueSize корректны

## Продвинутые кейсы
- Задача бросает исключение — Future пробрасывает
- Отмена задачи через Future.cancel()
- awaitTermination с таймаутом
- Большое количество задач на малый пул
- Потокобезопасность при конкурентных submit
