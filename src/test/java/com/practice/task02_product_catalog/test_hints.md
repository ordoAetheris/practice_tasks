# Подсказки для тестирования: ProductCatalogService

## Обязательно протестировать
- init() с корректными данными — товары привязаны к категориям
- isProductInCategory — товар принадлежит категории (true)
- isProductInCategory — товар НЕ принадлежит категории (false)
- getProductsByCategory — возвращает товары отсортированные по id
- Товар в нескольких категориях — находится в каждой из них
- Категория без товаров — пустой список
- Несуществующий productId или categoryId

## Продвинутые кейсы
- Большое количество товаров и категорий — проверка производительности
- Повторный вызов init() — корректное поведение (перезапись или ошибка?)
- Пустой список товаров или категорий при init()
- Товар без категорий
