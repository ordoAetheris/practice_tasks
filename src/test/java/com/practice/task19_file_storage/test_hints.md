# Подсказки для тестирования: FileStorageService

## Обязательно протестировать
- Создание файла и директории
- Дубликат имени в директории — исключение
- Перемещение файла
- Удаление файла
- Поиск по имени (подстрока)
- Рекурсивный размер директории

## Продвинутые кейсы
- Вложенные директории (3+ уровня)
- Одинаковые имена в разных директориях — допустимо
- Пустая директория — размер 0
