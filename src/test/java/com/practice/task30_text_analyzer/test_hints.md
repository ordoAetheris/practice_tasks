# Подсказки для тестирования: TextAnalyzerService

## Обязательно протестировать
- Частота слов — "hello world hello" -> {hello=2, world=1}
- Регистронезависимость — "Hello HELLO hello" -> {hello=3}
- Знаки препинания не часть слова — "hello, world!" -> {hello=1, world=1}
- Топ-N слов с корректной сортировкой
- Количество предложений — подсчёт '.', '!', '?'
- Средняя длина слова
- Уникальные слова — без дубликатов
- Слова длиннее N символов

## Продвинутые кейсы
- Пустой текст — пустые результаты, 0 предложений, 0.0 средняя длина
- Текст без слов (только цифры и знаки) — пустые результаты
- Многоточие "..." — сколько предложений?
- Текст с переносами строк — корректный подсчёт строк
- TopN с N больше количества уникальных слов — вернуть все
- Одинаковая частота слов — алфавитный порядок в topN
- Слова с дефисом "какой-то" — одно или два слова?
- Числа в тексте "abc123def" — как обрабатывать?
