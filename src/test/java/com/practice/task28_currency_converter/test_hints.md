# Подсказки для тестирования: CurrencyConverterService

## Обязательно протестировать
- Установка курса и проверка прямого и обратного курсов
- Конвертация суммы — результат = amount * rate
- Конвертация одинаковых валют — возвращается та же сумма
- Курс <= 0 — ожидать IllegalArgumentException
- Список доступных валют после добавления нескольких пар
- Получение курса для несуществующей пары — Optional.empty
- История курса после нескольких обновлений

## Продвинутые кейсы
- Лучший курс через промежуточную валюту: USD->EUR напрямую vs USD->GBP->EUR
- Обратный курс: если USD->EUR = 0.85, то EUR->USD = 1/0.85
- Обновление курса — старое значение в истории, новое — текущее
- Конвертация отрицательной суммы — ожидать исключение
- findBestRate когда нет ни прямого пути, ни промежуточного
- История отсортирована по дате (новые первыми)
- Одинаковые валюты в setRate — ожидать исключение
