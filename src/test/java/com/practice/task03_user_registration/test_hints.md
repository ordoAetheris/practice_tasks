# Подсказки для тестирования: UserRegistrationService

## Обязательно протестировать
- Успешная регистрация — пользователь создан с корректными полями
- Регистрация с пустым логином — IllegalArgumentException
- Регистрация с невалидным email (без @, без домена) — IllegalArgumentException
- Регистрация с дублирующимся логином — IllegalStateException
- Регистрация с дублирующимся email — IllegalStateException
- Поиск по логину — существующий и несуществующий
- Поиск по email — существующий и несуществующий
- Удаление существующего пользователя
- Удаление несуществующего пользователя — false
- Сортировка по дате регистрации

## Продвинутые кейсы
- Регистрация после удаления — логин и email снова доступны
- Email с разным регистром — считаются одинаковыми или нет?
- Логин с пробелами
- Автоинкремент id — корректная генерация
- Пустой список пользователей — getAllSortedByRegistrationDate возвращает пустой список
