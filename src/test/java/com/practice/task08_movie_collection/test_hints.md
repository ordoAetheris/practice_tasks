# Подсказки для тестирования: MovieCollectionService

## Обязательно протестировать
- Добавление фильма — успех
- Добавление фильма с дублирующимся id — ошибка
- Удаление существующего и несуществующего фильма
- Поиск по названию — подстрока, регистронезависимый
- Фильтр по жанру — фильм с несколькими жанрами найден по каждому
- Фильтр по году
- Фильтр по минимальному рейтингу
- Топ-N — корректная сортировка по рейтингу
- Средний рейтинг по жанру

## Продвинутые кейсы
- Топ-N где N больше количества фильмов
- Фильм с рейтингом ровно на границе (0.0, 10.0)
- Невалидные данные: рейтинг < 0 или > 10, пустое название
- Жанр без фильмов — средний рейтинг 0.0
- Комбинация фильтров (по жанру и рейтингу одновременно — через цепочку вызовов)
