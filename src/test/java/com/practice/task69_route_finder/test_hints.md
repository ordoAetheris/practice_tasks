# Подсказки для тестирования: RouteFinderService

## Обязательно протестировать
- findShortestPath между соседними городами
- findShortestPath выбирает путь с минимальным расстоянием (не обязательно с минимумом остановок)
- findFastestPath может отличаться от кратчайшего
- Односторонняя дорога: путь A->B есть, B->A нет
- Нет пути — возвращает Optional.empty()
- findAllPaths с maxStops корректно ограничивает глубину
- getReachableCities не включает недостижимые города

## Продвинутые кейсы
- Граф с циклами: Dijkstra не зацикливается
- findAllPaths не зацикливается на циклах (не посещает вершину дважды)
- findNearestCity исключает указанные города
- Добавление дороги с расстоянием <= 0 бросает исключение
- Большой граф: производительность Dijkstra приемлема
- Несколько дорог между одними городами (выбирается лучшая)
- getReachableCities: пограничное расстояние (ровно maxDistance)
