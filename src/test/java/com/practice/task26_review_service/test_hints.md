# Подсказки для тестирования: ReviewService

## Обязательно протестировать
- Добавление отзыва — проверить что создаётся со статусом ACTIVE
- Рейтинг вне диапазона 1-5 — ожидать IllegalArgumentException
- Повторный отзыв от того же пользователя на тот же продукт — ожидать IllegalStateException
- Мягкое удаление — статус переходит в DELETED
- Средний рейтинг — считается только по ACTIVE отзывам
- Сортировка отзывов по дате и рейтингу
- Топ-N продуктов по рейтингу

## Продвинутые кейсы
- Средний рейтинг продукта без ACTIVE отзывов (все удалены) — ожидать 0.0
- Отзывы пользователя включают все статусы (ACTIVE, MODERATED, DELETED)
- Топ продуктов при одинаковом рейтинге у нескольких продуктов
- Модерация уже удалённого отзыва — допустимо ли?
- Пустой текст отзыва — ожидать исключение
- Большое количество отзывов — корректность сортировки
- getTopProducts(0) или getTopProducts с N больше количества продуктов
