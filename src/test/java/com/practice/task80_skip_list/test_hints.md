# Подсказки для тестирования: SkipList

## Обязательно протестировать
- Вставка и получение по ключу
- Обновление значения существующего ключа
- Удаление элемента
- contains для существующего и несуществующего
- floor и ceiling
- range возвращает правильный диапазон
- size корректен после вставок/удалений

## Продвинутые кейсы
- Большое количество элементов — проверить что поиск работает
- Удаление несуществующего ключа
- floor/ceiling на граничных значениях
- range с пустым результатом
- Порядок элементов при обходе
