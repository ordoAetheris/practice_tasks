# Подсказки для тестирования: MarkAndSweepGC

## Обязательно протестировать
- Объект в корнях — не собирается
- Объект достижимый от корня через ссылки — не собирается
- Недостижимый объект — собирается
- gc() возвращает правильное количество собранных
- heapSize и liveCount корректны

## Продвинутые кейсы
- Циклические ссылки без связи с корнями — собираются
- Удаление корня делает поддерево мусором
- Повторный gc() без изменений — 0 собранных
- Цепочка ссылок: root -> A -> B -> C, удаление ссылки A->B
