# Подсказки для тестирования: VotingSystemService

## Обязательно протестировать
- Создание опроса с вариантами
- Голосование учтено в результатах
- Повторное голосование — исключение
- Голосование в закрытом — исключение
- Корректный подсчёт голосов

## Продвинутые кейсы
- Невалидный optionIndex
- Опрос без голосов
- Менее 2 вариантов при создании
- Просроченный опрос
