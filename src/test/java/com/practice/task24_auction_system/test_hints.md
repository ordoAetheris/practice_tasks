# Подсказки для тестирования: AuctionSystemService

## Обязательно протестировать
- Создание лота — статус ACTIVE, currentPrice == startPrice
- Ставка больше текущей цены — успешно, currentPrice обновляется
- Ставка <= текущей цены — ожидать исключение
- Продавец ставит на свой лот — ожидать IllegalStateException
- Ставка на завершённый лот — ожидать исключение
- Завершение аукциона со ставками — статус SOLD
- Завершение аукциона без ставок — статус EXPIRED
- Список активных лотов, история ставок, лоты продавца

## Продвинутые кейсы
- Создание лота с отрицательной или нулевой ценой
- Несколько ставок подряд от разных участников — currentPrice растёт
- Повторное завершение уже завершённого аукциона
- История ставок отсортирована по времени (новые первыми)
- Пустой список активных лотов после завершения всех
- Лоты продавца в разных статусах
