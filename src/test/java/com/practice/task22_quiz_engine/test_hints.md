# Подсказки для тестирования: QuizEngineService

## Обязательно протестировать
- Создание теста — проверить что вопросы сохранены корректно
- Начало попытки — проверить что startedAt заполнен, finishedAt == null
- Ответ на вопрос — проверить что ответ сохраняется в карте answers
- Завершение попытки — проверить подсчёт баллов (количество правильных ответов)
- Нельзя ответить после завершения попытки — ожидать IllegalStateException
- Нельзя начать попытку на тест без вопросов
- Статистика — средний балл и процент прохождения

## Продвинутые кейсы
- Создание теста с пустым списком вопросов — ожидать исключение
- Ответ на несуществующий вопрос в рамках попытки
- Повторный ответ на тот же вопрос (перезапись или ошибка?)
- Несколько попыток одного пользователя на один тест
- Статистика при отсутствии завершённых попыток
- Проверка процента прохождения при граничном значении 60%
- Завершение попытки без единого ответа — score = 0
