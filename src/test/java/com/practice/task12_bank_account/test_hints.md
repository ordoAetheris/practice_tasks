# Подсказки для тестирования: BankAccountService

## Обязательно протестировать
- Создание счёта и начальный баланс = 0
- Пополнение увеличивает баланс
- Снятие уменьшает баланс
- Снятие больше баланса — исключение
- Перевод: баланс отправителя уменьшается, получателя увеличивается
- Перевод между разными валютами — исключение
- История транзакций содержит все операции

## Продвинутые кейсы
- Пополнение и снятие нулевой или отрицательной суммы
- Перевод самому себе
- Несуществующий счёт
- История транзакций отсортирована по времени
- Множественные переводы — итоговые балансы сходятся
