# Подсказки для тестирования: AutocompleteService

## Обязательно протестировать
- addWord + contains — слово найдено
- search по префиксу — возвращаются только слова с этим префиксом
- searchTopK — возвращаются K слов с наибольшим весом
- Сортировка по весу (убывание), при одинаковом весе — алфавит
- addText — слова добавляются с весом равным частоте
- removeWord — слово больше не находится contains и search
- Регистронезависимость — "Hello" и "hello" — одно слово

## Продвинутые кейсы
- Повторное добавление слова — вес суммируется
- Пустой префикс — возвращает все слова
- Префикс, которого нет — пустой результат
- searchTopK с k > количества слов — возвращает все
- addWord с пустым словом — ожидать исключение
- addWord с weight <= 0 — ожидать исключение
- Удаление слова, являющегося префиксом другого — другое слово сохраняется
- addText с текстом без слов — ничего не добавляется
- contains для префикса (не полного слова) — false
