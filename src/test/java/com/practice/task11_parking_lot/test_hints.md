# Подсказки для тестирования: ParkingLotService

## Обязательно протестировать
- Парковка MOTORCYCLE на любое место (REGULAR, HANDICAPPED, LARGE)
- Парковка CAR только на REGULAR или HANDICAPPED
- Парковка TRUCK только на LARGE
- Отказ парковки при отсутствии подходящих мест
- Выезд по билету, место освобождается
- Поиск машины по номеру
- Подсчёт свободных мест корректен после парковки/выезда

## Продвинутые кейсы
- Полная парковка — все места заняты
- Повторная парковка на освобождённое место
- Несколько этажей, распределение по этажам
- Поиск несуществующей машины
- Выезд по невалидному билету
