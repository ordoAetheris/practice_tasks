# Подсказки для тестирования: VendingMachine

## Обязательно протестировать
- Полный цикл: insertCoin -> selectProduct -> dispense
- Корректный расчёт сдачи (жадный алгоритм)
- Недостаточно средств — selectProduct бросает исключение
- Товар закончился — selectProduct бросает исключение
- returnCoins возвращает внесённые монеты, состояние → IDLE
- getBalance корректен после нескольких insertCoin
- Состояния: операции в неправильном состоянии бросают исключение

## Продвинутые кейсы
- Сдача из нескольких номиналов (жадный алгоритм: максимально крупные)
- Точная сумма — сдача пустая
- refill увеличивает количество товара
- Несколько покупок подряд — количество уменьшается
- insertCoin в состоянии COIN_INSERTED (добавление к балансу)
- selectProduct несуществующего товара — исключение
- Переход состояний: IDLE → COIN_INSERTED → DISPENSING → IDLE
