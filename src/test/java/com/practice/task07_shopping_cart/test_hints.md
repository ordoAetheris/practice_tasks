# Подсказки для тестирования: ShoppingCartService

## Обязательно протестировать
- Создание корзины — пустая, с уникальным id
- Добавление товара — товар появляется в корзине
- Повторное добавление того же товара — количество увеличивается
- Удаление товара из корзины
- Изменение количества товара
- Подсчёт суммы корзины (несколько товаров с разными ценами и количествами)
- Очистка корзины
- Добавление с количеством <= 0 — ошибка
- Добавление товара с ценой <= 0 — ошибка

## Продвинутые кейсы
- Несколько корзин одновременно — независимы друг от друга
- Подсчёт суммы пустой корзины — 0
- Удаление товара из пустой корзины — ошибка
- Операции с несуществующей корзиной — ошибка
- Точность BigDecimal при расчёте суммы (копейки)
