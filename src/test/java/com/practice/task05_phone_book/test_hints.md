# Подсказки для тестирования: PhoneBookService

## Обязательно протестировать
- Добавление контакта — успешный сценарий
- Добавление контакта с пустым именем — ошибка
- Добавление контакта с уже существующим номером — ошибка
- Добавление номера к существующему контакту
- Удаление номера — успешное
- Удаление последнего номера — ошибка (контакт должен иметь хотя бы один номер)
- Поиск по имени — подстрока, без учёта регистра
- Поиск по номеру — точное совпадение
- Получение всех контактов — сортировка по имени

## Продвинутые кейсы
- Несколько контактов с похожими именами — поиск возвращает все
- Контакт с email = null
- Добавление одного и того же номера к тому же контакту
- Удаление номера у несуществующего контакта
- Пустая телефонная книга — все методы поиска возвращают пустые результаты
