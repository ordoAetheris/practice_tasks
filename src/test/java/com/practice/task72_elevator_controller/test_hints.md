# Подсказки для тестирования: ElevatorController

## Обязательно протестировать
- requestElevator + step: лифт приезжает на вызванный этаж
- selectFloor + step: лифт доезжает до выбранного этажа
- SCAN: лифт не разворачивается, пока есть запросы в текущем направлении
- SCAN: лифт разворачивается, когда запросов в текущем направлении нет
- getStatus возвращает корректное состояние всех лифтов
- IDLE состояние когда нет запросов

## Продвинутые кейсы
- getOptimalElevator: лифт в нужном направлении проезжающий мимо — приоритет
- getOptimalElevator: IDLE лифт ближе — предпочтителен
- Несколько лифтов: запросы распределяются между ними
- Лифт обслуживает промежуточные этажи по пути (подбирает/высаживает)
- Вызов за пределами допустимых этажей — обработка ошибки
- Лифт на крайнем этаже корректно разворачивается
- Множество одновременных запросов обрабатываются без потерь
