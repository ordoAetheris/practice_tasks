# Подсказки для тестирования: FlightBookingService

## Обязательно протестировать
- Бронирование на рейс с местами
- Бронирование на полный рейс — исключение
- Отмена бронирования, место освобождается
- Повторная отмена — исключение
- Поиск рейсов по маршруту
- Поиск рейсов по дате
- Подсчёт свободных мест после бронирования/отмены

## Продвинутые кейсы
- Несколько бронирований одного пассажира на разные рейсы
- Бронирование после отмены — место снова доступно
- Рейсы без бронирований
- Несуществующий рейс/бронь
