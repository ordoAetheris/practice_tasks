# Подсказки для тестирования: RecommendationEngineService

## Обязательно протестировать
- addRating с оценкой вне диапазона [1, 5] бросает исключение
- computeSimilarity: идентичные оценки = 1.0
- computeSimilarity: нет общих товаров = 0.0
- getRecommendations не включает уже оценённые товары
- getRecommendations возвращает не более topN элементов
- getSimilarItems находит товары с похожими паттернами оценок

## Продвинутые кейсы
- computeSimilarity: противоположные оценки дают отрицательное значение
- Пользователь без оценок — пустые рекомендации
- Обновление оценки (повторный addRating) — перезаписывает старую
- getPersonalizedFeed комбинирует оба подхода
- Один товар оценён одним пользователем — getSimilarItems пуст (недостаточно данных)
- Большое количество пользователей и товаров — корректность cosine similarity
- Рекомендации отсортированы по predictedScore
